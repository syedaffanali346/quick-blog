# Quick Blog App 🚀

A full-stack blog application built using the **MERN Stack** with **AI integration**.

🔗 **Live Demo:**
https://quick-blog-app-tau.vercel.app/

---

## 📌 Features

* 📝 Create, edit, and delete blogs
* 🔍 Search blogs
* 📂 Category-based organization
* 🤖 AI-powered blog description generator (Gemini AI)
* 👨‍💼 Admin panel for managing content
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Deployment:** Vercel
